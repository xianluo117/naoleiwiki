## 👋 欢迎新朋友

类脑ΟΔΥΣΣΕΙΑ 是 **SillyTavern（酒馆）最大的中文 Discord 交流社区**，汇聚技术探索者与自由创作者。

::: tip 💡 第一次来？
建议先阅读 [基础规则](/community-rules/basic-rules)，了解社区行为边界，然后去各个分区探索你感兴趣的内容！
:::

## 🌟 社区亮点

| 方向             | 说明                                                    |
| ---------------- | ------------------------------------------------------- |
| 🔞 **创作自由**   | 以成人向（R18）内容为主导，鼓励自由、开放地讨论 AI 创作 |
| 🚫 **纯粹非盈利** | 完全非商业化，严禁牟利、广告与引流                      |
| 🛠️ **技术探索**   | 大量 LLM、API、酒馆前端与插件的技术爱好者聚集地         |

## 📂 内容分区一览

- 🃏 **角色卡区** — 核心产出地，包含大量原创角色卡（男性向 / 女性向 / 全性向及世界书）
- 📦 **资源区** — 分享预设、破限（JB）、界面美化与实用插件
- 📚 **教程与技术** — 从新手入门到进阶实战的教程沉淀
- 🏆 **赛事与活动** — 定期举办主题创作活动，激发创作灵感
