类脑ΟΔΥΣΣΕΙΑ 是 SillyTavern（酒馆）最大的中文 Discord 交流社区，也是一个汇聚技术探索者与自由创作者的“绅士绿洲”。

## 社区简介

### 核心文化与精神

- 🔞 **NSFW 主导与创作自由**：以成人向（R18）内容为主导，在不违反社区准则的前提下，鼓励自由、开放地讨论成人 AI 创作与相关内容。
- 🚫 **纯粹非盈利**：社区完全非商业化，严禁牟利、广告、引流与诈骗行为，维护纯粹创作氛围。
- 🛠️ **技术探索**：聚集大量热衷于折腾 LLM、API、酒馆前端与插件的技术爱好者。

### 精彩内容分区

- 🃏 **角色卡区**：核心内容产出地，包含大量原创角色卡（男性向、女性向、全性向及世界书）。
- 📦 **资源区**：分享 SillyTavern 预设、破限（JB）、界面美化与实用插件。
- 📚 **教程与技术**：从新手入门到进阶实战的教程沉淀。
- 🏆 **赛事与活动**：定期举办主题创作活动，激发创作灵感。
