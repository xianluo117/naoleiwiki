## 🌌 欢迎来到类脑

**类脑ΟΔΥΣΣΕΙΑ** 是 SillyTavern（酒馆）最大的中文 Discord 交流社区，也是一个汇聚技术探索者与自由创作者的“绅士绿洲”。

### 🧬 核心文化

::: tip 🔞 NSFW 主导与创作自由
以成人向（R18）内容为主导，在严格遵守社区准则的前提下，我们鼓励自由、开放地讨论 AI 创作与相关技术。这里是思想碰撞的自由港。
:::

::: warning 🚫 纯粹非盈利
社区坚持**完全非商业化**运作。我们严禁任何形式的牟利、广告投放、恶意引流与诈骗行为，只为维护那份最纯粹的创作热情。
:::

### 🚀 精彩内容分区

| 分区             | 描述                                                   |
| :--------------- | :----------------------------------------------------- |
| **🃏 角色卡区**   | 核心内容产出地，汇聚海量原创角色卡（全性向/世界书）。  |
| **📦 资源区**     | 分享 SillyTavern 预设、破限 (JB)、界面美化与实用插件。 |
| **📚 教程与技术** | 从零基础入门到硬核进阶实战的知识沉淀。                 |
| **🏆 赛事与活动** | 定期举办主题创作活动，点燃你的创作灵感。               |
