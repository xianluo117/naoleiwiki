## 学习路线

建议按照以下顺序学习，从零开始掌握酒馆的使用：

| 顺序  | 教程                 | 说明                                 |
| :---: | -------------------- | ------------------------------------ |
|   1️⃣   | 什么是 SillyTavern？ | 了解酒馆是什么、能做什么             |
|   2️⃣   | 安装与部署           | 在你的电脑上安装并运行酒馆           |
|   3️⃣   | 连接 API             | 配置 AI 模型接口，让酒馆能和 AI 对话 |
|   4️⃣   | 导入角色卡           | 下载和导入你的第一个角色卡           |
|   5️⃣   | 开始对话             | 开始你的第一次 AI 角色扮演对话       |
|   6️⃣   | 进阶设置             | 参数调优、预设选择、界面美化等       |
