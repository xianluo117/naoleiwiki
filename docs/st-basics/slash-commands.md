# ⌨️ ST 斜杠命令

这些命令需要在聊天输入框中输入，并按发送键来使用。

::: tip 💡 提示
输入 `/help` 可以获取更多帮助信息。
:::

---

## 常用命令一览

### 获取帮助

| 命令            | 说明                         |
| --------------- | ---------------------------- |
| `/help slash`   | 获取斜杠命令帮助             |
| `/help macros`  | 获取宏相关帮助               |
| `/help format`  | 获取支持的 Markdown 语法帮助 |
| `/help hotkeys` | 获取快捷键帮助               |

### 消息管理

| 命令                          | 说明                                         |
| ----------------------------- | -------------------------------------------- |
| `/trigger`                    | 触发正常生成（相当于点击"发送"按钮）         |
| `/hide (specified message)`   | 隐藏一条或数条消息，隐藏的内容不会发送给 LLM |
| `/unhide (specified message)` | 取消隐藏一条或数条消息                       |
| `/cut (specified message)`    | 从聊天中删除指定的消息或连续片段             |
| `/chat-jump (index)`          | 跳到指定楼层                                 |

---

## 参数说明

### `specified message`（消息编号/范围）

- **单条消息**：直接填写消息编号，如 `/hide 5`
- **消息范围**：使用 `start-finish` 格式，表示范围内所有消息（包含始末），如 `/hide 0-100`

::: warning ⚠️ 注意
开始和结束楼层 **不可超出** 当前已有的楼层范围，否则命令无效。
:::

### `/cut` 命令的特殊能力

`/cut` 支持"从上向下"的删除，这是通常的删除操作做不到的。适合批量清理前文消息。

---

## 实际用例

### 总结前文时的常见操作

```
/hide 0-100        # 先隐藏前 100 条消息
/unhide 0-50       # 分段显示前 50 条，让 AI 总结
/hide 0-50         # 总结完毕后再次隐藏
/unhide 51-100     # 显示下一段，继续总结
```
