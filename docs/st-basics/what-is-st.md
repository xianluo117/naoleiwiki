# 🤔 什么是酒馆（SillyTavern）

## 简介

**SillyTavern**（简称 **ST**，中文社区常称为"酒馆"）是一个前端界面，允许你与 LLM（大语言模型）、AI 生图、TTS 语音合成等服务进行简单便捷的交互。

它本质上是一个运行在本地的 Web 应用，通过浏览器访问，为角色扮演、创意写作、AI 对话等场景提供了丰富的功能支持。

## 版本信息

| 信息             | 说明                           |
| ---------------- | ------------------------------ |
| **最新版本**     | 1.15.0                         |
| **推荐稳定版**   | 1.13.4（社区公认较稳定的版本） |
| **最低建议版本** | 1.13.4                         |

::: warning 版本提醒
低于 **1.13.4** 的版本**不推荐使用**——它们缺少运行最新社区扩展所必需的功能，可能导致兼容性问题。
:::

## 系统要求

- 需要能运行 **Node.js 18** 或更高版本的设备
- **Windows 7 及更早版本不支持**（无法安装所需版本的 Node.js）
- 支持的操作系统：Windows 10+、Linux、MacOS、Android（通过 Termux）

::: tip iOS 用户
iOS 系统**不支持**本地安装 SillyTavern，但可以通过网络连接到其他设备（如云服务器、局域网内的电脑）上运行的 ST 来使用。
:::
