# 💬 Discord 相关问题

本页面收录在类脑 Discord 社区中的常见操作问题及解决方案。

---

## 频道与权限

### 我想看的频道找不到了！

1. 在频道列表最上方，点击「浏览频道」
2. 在弹出的列表中，找到你想看的频道，勾选它右侧的方框，它就会固定显示在你的频道列表里了。反之，取消勾选可以隐藏不常用的频道
3. ⚠️ **特别注意**：`#深渊区` 和 `#R18G绘图` 等频道，需要先去「深渊之门」频道领取特定身份组后才能看到

---

### 为什么我不能在某些频道说话？

这是由于类脑的 **缓冲区机制**：

| 阶段         | 状态       | 说明                                                 |
| ------------ | ---------- | ---------------------------------------------------- |
| 新成员加入   | 缓冲区     | 120 小时内无法发言                                   |
| 通过道馆挑战 | 高级缓冲区 | 可在新人答疑区和开帖区发言，剩余缓冲时长变为 72 小时 |
| 缓冲计时结束 | 已验证     | 允许在绝大多数频道发言                               |
| 完成道馆挑战 | 已验证     | 可立刻离开缓冲区，获得已验证身份组                   |

## 常用操作

### 怎么回到帖子的最上面？（回顶）

**电脑桌面端**：有直接回到顶部的按钮

**手机 APP**：
- 在聊天框发送命令 `/回顶`，机器人会给你发送一个只有你能看到的返回顶部按钮，点击即可
- 对最下面一条消息长按，选择「APP」，再点击「回到顶部」旁边的「发送」，同样可以得到回顶按钮

**还在缓冲区的用户**：由于缓冲区无法在资源区发言，需要使用回顶区（答疑区的子区）

### 怎么下载角色卡？

1. 在 Discord 类脑的频道列表中找到角色卡区（被分成了多个频道），点进去就能看到很多角色卡
2. 卡的作者会指路哪一层的图片就是卡，下载这张图片的 **原图**（应该是 PNG 格式）就可以导入酒馆了
3. 下载原图的方法：
   - **电脑桌面端**：直接右键点击图片，选择「保存图片」
   - **手机 APP**：点开大图长按，选择「保存图片」
   - **网页端**：⚠️ 必须点开大图后，选择右上角的「在浏览器中打开」，然后在新打开的页面中保存图片

::: warning ⚠️ 注意
网页端直接保存可能会得到压缩后的图片（非 PNG），导致角色卡元数据丢失而无法导入。
:::
