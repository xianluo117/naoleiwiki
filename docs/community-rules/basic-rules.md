# ✅ 基础规则

::: info 📌 适用范围
本规则适用于类脑ΟΔΥΣΣΕΙΑ 社区全体成员，涵盖所有频道与互动场景。
:::

---

## 1. 合法合规

- 🚫 **严禁发布违法内容**，包括但不限于涉及未成年人的不当内容、暴力恐怖内容等
- 🚫 **严禁任何形式的诈骗、引流、恶意营销**
- 遵守 Discord 平台的 [服务条款](https://discord.com/terms) 和 [社区准则](https://discord.com/guidelines)

---

## 2. 非商业化与反牟利

- 社区为 **非营利交流空间**，所有资源分享均以自愿为原则
- 🚫 **禁止** 在社区内进行未经许可的商业推广与收费导流
- 🚫 **禁止** 以任何形式向社区成员收费或售卖内容

::: tip 💡 提示
如果你发现有人在社区内进行商业行为，请及时向管理员反馈。
:::

---

## 3. 尊重与秩序

- 🤝 **尊重他人创作成果**，禁止恶意攻击、骚扰、歧视
- 💬 友善交流，避免无意义的争吵与人身攻击
- ⚖️ 出现争议时，优先走 **反馈与管理协商流程**，不要私下升级冲突

---

## 4. NSFW 讨论边界

- 社区允许 **成人向（R18）创作讨论**，这是社区文化的一部分
- 但必须遵守 **平台政策与社区边界**：
  - ✅ 允许：AI 角色创作讨论、预设分享、技术交流
  - 🚫 禁止：传播违规或高风险内容（如真人色情、未成年相关等）

::: danger ⛔ 红线警告
触及以下内容将被 **立即永久封禁**，不予申诉：

- 涉及未成年人的任何不当内容
- 真实人物的非自愿色情内容
- 任何违反当地法律的内容
:::
