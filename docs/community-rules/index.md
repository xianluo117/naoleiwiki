## 核心原则

社区规则围绕以下 **四大核心原则** 制定：

1. **合法合规** — 所有内容不得违反法律法规
2. **非商业化** — 社区是纯粹的非盈利交流空间
3. **相互尊重** — 尊重创作者与每一位成员
4. **内容安全** — 在允许范围内保障内容质量

::: warning ⚠️ 违规须知
违反社区规则将视严重程度予以 **警告、禁言或永久封禁** 处理。如有争议，请通过管理协商流程反馈。
:::
