# 📦 子区规则

::: info 📌 适用范围
本规则主要适用于社区内的 **资源区、角色卡区** 等内容发布频道。
:::

---

## 1. 发布要求

- 📝 资源发布需注明 **用途、来源与适用范围**
- 📎 尽量附带基础说明：
  - 安装方式
  - 使用方法
  - 兼容性信息（如适配的酒馆版本）

::: tip 💡 好的发布习惯
一个完善的资源帖应该包含：**标题 + 简介 + 使用说明 + 截图/预览 + 下载链接**。这能帮助其他成员更快地理解和使用你的资源。
:::

---

## 2. 版权与授权

- 🚫 **禁止搬运** 未经授权的付费资源或私有内容
- 📌 引用他人作品需 **标注原作者与来源链接**
- 如果你是原创作者，建议注明授权方式（如自由使用、禁止二次分发等）

---

## 3. 安全与质量

- 🛡️ **禁止上传** 包含恶意脚本、木马或可疑文件的资源
- ✅ 建议发布前进行 **可用性测试**，确保：
  - 链接有效
  - 文件完整无损坏
  - 功能可正常使用

::: danger ⛔ 安全红线
上传恶意文件将被 **立即封禁**。如发现可疑资源，请立即向管理员举报。
:::

---

## 4. 反馈与维护

- 💬 发布者应 **关注评论反馈** 并适度维护
- 🔄 失效资源应及时：
  - 修复链接
  - 下架资源
  - 或标注「已失效」状态

---

## 发布规范速查

| 项目 | 要求                           |
| ---- | ------------------------------ |
| 标题 | 清晰描述资源名称和类型         |
| 来源 | 原创标注「原创」，搬运标注出处 |
| 说明 | 附带安装/使用说明              |
| 版权 | 不得搬运付费/未授权内容        |
| 安全 | 不得包含恶意代码               |
| 维护 | 关注反馈，及时更新             |
